Detailed Description

template<class _MetaFeatureType, class _ScoreContextType, class _ActionType>
class ZuoPar::FeaturePointwiseParameterMap< _MetaFeatureType, _ScoreContextType, _ActionType >

FeaturePointwiseParamMap is the class use to store the model parameters of the feature. One FeaturePointwiseParamMap maintains several Features. These features are extracted from the same feature template.

Map [ ActionFeature ] = Parameter
