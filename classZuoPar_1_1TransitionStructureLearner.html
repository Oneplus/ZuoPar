Train the parameters with passive aggressive algorithm. According to K. Crammer et. al (2006), the one-best MIRA (also named passive aggressive) has close form solution on derivation.

w_t+1 = w_t + * x_t, where = l_t / || x_t ||^2

For the learning guided search problem, we set the number of wrongly actions and ther score as loss l_t
